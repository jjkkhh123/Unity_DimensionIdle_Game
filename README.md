# 반물질 키우기 (Antimatter Idle)

Antimatter Dimensions에서 영감을 받은 방치형 증분 게임입니다.

## 게임 소개

8개의 차원을 통해 반물질을 무한대까지 생산하는 것이 목표입니다. 각 차원은 하위 차원을 생산하고, 프레스티지와 다양한 업그레이드를 통해 생산량을 극대화하세요.

## 주요 기능

- **8차원 생산 시스템** - 각 차원이 하위 차원을 생산
- **틱스피드 업그레이드** - 전체 생산 속도 증가
- **Dimension Boost** - 차원 해금 및 영구 배수 획득
- **프레스티지** - 리셋 후 강력한 업그레이드 구매
- **상점** - 프리미엄 부스트 아이템
- **오프라인 진행** - 접속하지 않아도 자원 축적

## 기술 스택

- **엔진**: Unity 6 (6000.1.2f1)
- **UI**: UI Toolkit (UXML/USS)
- **대형 숫자 처리**: BigDouble (1e308+ 지원)

## 게임 목표

| 단계 | 목표 |
|------|------|
| 초반 | 8차원 해금 |
| 중반 | 프레스티지 업그레이드 수집 |
| 엔드게임 | 반물질 1.79e308 (Infinity) 도달 |

## 조작법

- **구매**: 각 차원/업그레이드 버튼 클릭
- **Max 구매**: Max 버튼으로 최대 구매
- **탭 전환**: 하단 탭으로 화면 이동
